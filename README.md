# bangla-keymap
ফোনেটিক বেজড বাংলা কিবোর্ড লে-আউট । এখান থেকে ডাউনলোড করে ইনষ্টল করে নিন। এটি ইনষ্টল করার পর আপনার উইন্ডোজ কম্পিউটারে এই ছবির মতও একটি অপসণ দেখতে পাবেন। এখান থেকে বাংলা অপশনটি সিলেক্ট করে নিলে আপনার কম্পিউটারটি বাংলা লেখার জন্য তৈরি হয়ে যাবে।
